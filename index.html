NodeJS

NodeJS is a very powerful JavaScript-based framework/platform built on Google Chrome's JavaScript V8 Engine. It is used to develop I/O intensive web applications like video streaming sites, single-page applications, and other web applications. NodeJS is open source, completely free, and used by thousands of developers around the world.
